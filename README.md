# PlanSync - Academic Planner

A comprehensive academic planner application that helps students manage their class schedules, homework, tasks, notes, and reminders all in one place.

## Features

### 📊 Dashboard
- Overview of all academic activities
- Quick stats showing classes today, pending tasks, notes count, and upcoming reminders
- Quick access to recent notes, today's classes, and upcoming tasks

### 📅 Class Schedule
- Weekly schedule view (Monday - Saturday)
- Add, edit, and delete classes
- Color-coded classes for easy identification
- Auto-color suggestions based on class name
- Location and time management
- "Now" button for quick time entry

### ✅ Homework & Tasks
- Create and manage tasks with priority levels (Low, Medium, High)
- Track task status (To Do, In Progress, Done)
- Due date tracking with "Today", "Tomorrow", "Overdue" indicators
- Filter tasks by status
- Sort by due date, priority, or title
- Grid/List view toggle

### 📝 Notes
- Create and organize notes with rich text
- Tag system for categorization
- Search notes by title or content
- Edit and delete notes
- Note preview with content truncation

### 🔔 Reminders
- Personal reminders with date and time
- Automatic reminders for:
  - Today's classes
  - Tomorrow's classes
  - Upcoming tasks
- Quick time presets (+15 min, +30 min, +1 hour, +2 hours, Tomorrow)
- Search functionality
- Today/All view toggle

## Technology Stack

### Backend
- **Laravel 12.x** - PHP Framework
- **MySQL** - Database
- **Eloquent ORM** - Database ORM

### Frontend
- **Vue 3** - JavaScript Framework
- **Vite** - Build tool and development server
- **Bootstrap 5** - CSS Framework
- **Font Awesome 6** - Icons
- **Axios** - HTTP client

### Development Tools
- **Composer** - PHP dependency manager
- **NPM** - Node package manager
- **Git** - Version control

## System Requirements

- PHP >= 8.1
- Composer
- Node.js >= 18
- MySQL >= 5.7
- Git (optional)
